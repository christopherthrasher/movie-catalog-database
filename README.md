## Movie Catalog Database System

A Python command-line application that manages a movie catalog using a PostgreSQL database. The system supports browsing movies, genre-based searches, user accounts, movie ratings, and rating aggregation.

### Features
- CSV-based movie data ingestion
- Relational PostgreSQL schema
- SQL-based querying and filtering
- User accounts and movie ratings
- Command-line interface

### Technologies
- Python 3
- PostgreSQL
- psycopg3
- SQL

### Running the Project
This project requires a local PostgreSQL database.

High-level steps:
1. Create a PostgreSQL database
2. Run the schema SQL to create tables
3. Import movie data from CSV
4. Run the CLI application

Detailed setup and run instructions are provided in `instructions.txt`.
